Queue module for Zend Framework 3.

Installation
------------

The preferred way to install this extension is through composer.

Either run

```
$ php composer.phar require bupy7/zf-queue "*"
```

or add

```
"bupy7/zf-queue": "*"
```

to the require section of your composer.json file.
